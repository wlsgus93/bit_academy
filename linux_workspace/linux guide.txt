sudo apt install net-tools -> ifconfig 등의 넷 도구들을 사용 가능하게 해줌

ifconfig -> inet 나오는 ip주소 firefox 에서 입력 후 이상 없을 시 넷 연결 완료
                    192.168.9.128:80

default port no : 80

/etc/apache2 ports.conf
